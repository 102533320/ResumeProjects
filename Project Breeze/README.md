# Breeze


## Run mobile app

```
cd mobile
npm i
npm start
```

## Run admin app

```
cd admin
npm i
npm start
```
visit admin app at `http://localhost:3000/`

## Run server

```
cd server
npm i
docker-compose up
npm start:dev
```
visit playground at `http://localhost:5000/graphql`

## Run website

open ```website/index.html```
